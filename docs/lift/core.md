# Core

## Gym / Weight Room

<details>
<summary>Machine and cable core exercises</summary>

### Cable Exercises

- Cable Woodchop (high to low, low to high)
- Cable Crunch (kneeling)
- Pallof Press (anti-rotation)

### Machine Exercises

- Ab Crunch Machine
- Torso Rotation Machine

### Bar Exercises

- Hanging Leg Raises
- Hanging Knee Raises
- Toes to Bar

### Ab Wheel

Roll out from kneeling position, extending arms while keeping core tight.
